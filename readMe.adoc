* CCP commands understood by the NXP ECU code:
 ** CCP_DIAG_SERVICE: Service 0x08, used to start the application after
    programming. No response on this message is received any more.
 ** CCP_SET_MTA: Set adress for subsequent CCP_CLEAR_MEMORY, PROGRAM_6 and
    CCP_UPLOAD commands.
 ** CCP_UPLOAD: Read memory for verify. Restricted to the configured ROM
    blocks. 
 ** CCP_PROGRAM6: Download 6 bytes to flash. They are collected in a
    buffer and programmed synchronously when the buffer is full or a flash
    sector boundary is reached. ccp_process() is blocking during
    programming.
 ** CCP_CLEAR_MEMORY: Erasure of an area beginning at MTA. The length is
    element of the command. It is unclear yet, whether the host needs to
    use correct flash sector boundaries (and thus use this cmmand
    repeatedly) or if all sectors are erased, which overlap with the
    demanded area.
