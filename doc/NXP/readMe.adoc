This folder contains some relevant extracts from the NXP flash boot loader
for MPC5777C and MPC5775B/E. An inspection of these files tells, which CCP
commands are used, and in which way, for programming the MCU. The flash
tool under development, `winFlashTool`, is designed to use the same
commands in order to be compatible with the existing NXP flash boot
loaders.

All aspects, which are relevant to the development of the flash tool will
be same for most or all other NXP MCUs, which can be flashed with the NXP
flash tool `RAppID_BL` plus the `*.rbf` or `*.rba` files. Which means that
our flash tool `winFlashTool` should be applicable for all of these MCUs,
too.
